# LeapNest Legal Documents

This repository contains the public Terms of Service and Privacy Policy documents for the LeapNest mobile application.

## Purpose

These legal documents are hosted publicly on GitHub Pages to provide easy access for users and to comply with legal requirements for mobile applications. The documents are designed to be displayed within the LeapNest app's webview.

## Documents

- **[Privacy Policy](./privacy-policy.html)** - Explains how LeapNest collects, uses, and protects user information
- **[Terms of Service](./terms-of-service.html)** - Outlines the terms and conditions for using the LeapNest application

## Hosting

These pages are hosted on GitHub Pages and can be accessed directly via their URLs:
- Privacy Policy: `https://aalbrand84.github.io/homeLeap-Policies/privacy-policy.html`
- Terms of Service: `https://aalbrand84.github.io/homeLeap-Policies/terms-of-service.html`

## Updates

When legal documents need to be updated:
1. Edit the respective HTML file
2. Update the "Last Updated" date in the document
3. Commit and push changes to the repository
4. Changes will be automatically reflected on GitHub Pages
